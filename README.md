# docs-map-vis

Map-based Docs Visualization


注：代码写得有点烂，主要有两个问题，一是性能不高，速度慢；二是文档投影到二维平面后，不能有坐标完全相同的两个投影节点，如果有坐标相同的投影节点，可视化结果可能会出错（颜色映射出现问题）。
